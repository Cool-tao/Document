### 数据类型
#### 基本数据类型 -  整型  
bit  1位，0 1  
boolean 1位，  0 1 
byte  8位，-128 ~ 127  
short  16位，-32768 ~ 32767  
int   32位，-2 147 483 648 ~ 2 147 483 647  
char  16位， 0 ~ 65535  
long  64位，-9 223 372 036 854 775 808 ~ 9 223 372 036 854 775 807  
基本数据类型 -  浮点型  
float  32位，  
double  64位，

#### 操作码    
returnAddress  
指向一条虚拟机指令的操作码；    

#### 应数据类型  
类 类型  class type  
数组类型  array type  
接口类型  interface type  
引用数据类型，有一个特殊值 null； 当一个引用，不指向任何对象的时候，TA的值就用null表示；
